Project information

  • Category: Medical Imaging
  • Client: University of Dayton
  • Project date: 23 June, 2020
  • Project URL: SPIE Link

Abstract

Diabetic retinopathy (DR) is the leading cause of blindness, affecting over 93 million people. This research aims to automate DR detection and grading using a hybrid machine learning architecture applied to a dataset of 3,662 retinal images. The approach combines transfer learning-based models such as AlexNet, VGG16, ResNet, and others, with convolutional neural networks (CNNs) and principal component analysis for dimensionality reduction, followed by support vector machine classification. The system achieved 98.4% accuracy in DR detection and 96.3% in severity grading, demonstrating robustness with limited data and offering a proof-of-concept for future clinical applications.
